# Overview

What is Firebase Manager?

> Firebase Manager is a tool esier to manage, access, maintenanace Firebase features.

What can Firebase Manager do?

> Firebase Manager can prepare all Firebase features at only one library.

Why Firebase Manager?

> Firebase Manager can access on abstract, maybe we can access other platform by same interface

> Firebase Manager access database by reference style, you don't need use any long path string in your code.

> Firebase Manager can auto mainte between the Firebase storage and your local storage.<br>
> Firebase Manager also can mainte the file relate with md5, you can keep a string as your file.

> Firebase Manager supported google and apple account login, we can call login method just by one method.(we will support other login method)<br>
> Firebase Manager include a simple login page, if you want to test your login algorithm, you can use it by quicky.
